# 🎭 Emoji Sentiment Detector (Google Colab)

A beginner-friendly hybrid emoji detector using:
- Hugging Face sentiment analysis 🤗
- Keyword-based emotion detection 💡
- Fun emoji results based on mood 😊😢😡
